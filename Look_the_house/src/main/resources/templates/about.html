This is 'Look the house' project.By this project any person in any country can provide rent office,mess,flat,house,hotel,garage,commercial-space,others and get rent from any place.
You can visit this site as an general user,if any person chooses a service then he/she can contact to the owner and get service easily.If a person want to rent a service then at first it should registration then he/she is allowed to add a service.
In home page if you click a country name then the services is shown which are provides by that country. If we click any picture then it is shown all same categories that are provided by various country. If you are a rent provider then you can edit your services.
